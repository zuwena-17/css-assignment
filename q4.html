Tutorial 1: Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a markup format consisting of various HTML elements.

Basic HTML Structure

Every HTML document consists of the following basic structure:

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Page Title</title>
                    </head>
                    <body>
                        <h1>Heading</h1>
                        <p>Paragraph</p>
                    </body>
                    </html>

HTML Elements

HTML elements are the building blocks of HTML pages. They are represented by tags enclosed in angle brackets, and they define the structure and content of the web page.

Common HTML elements include headings, paragraphs, links, images, lists, tables, forms, and more.

HTML Terminology

HTML
HyperText Markup Language - The standard markup language for creating web pages.
Element
A part of a webpage that is defined by tags and contains specific content or serves a specific purpose.
Tag
An HTML code that specifies the structure and formatting of content within a webpage.
